## 📌 Descripción

Este proyecto es una aplicación tipo CRUD (Create, Read, Update, Delete) enfocada en la gestión de clientes.

Actualmente incluye:

* Agregar clientes
* Mostrar lista de clientes
* Eliminar clientes

El frontend está desplegado en línea y el backend fue desarrollado con Flask para uso local.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Python (Flask)

---

## ⚙️ Funcionamiento

* El usuario escribe un nombre en el input
* Al hacer clic en "Agregar", el cliente se añade a la lista
* Cada cliente tiene un botón para eliminarlo
* La interfaz es responsive (adaptable a celular y PC)

---

## 📌 Nota importante

Este proyecto está adaptado para mostrarse en GitHub Pages, por lo que:

* El frontend funciona completamente en línea
* El backend con Flask funciona de manera local (no desplegado)
